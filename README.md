# MusicGame Result Calcuration

音ゲーのリザルトを集計するアプリです。

## 説明

Discordのスラッシュコマンドを使ってリザルト画像からOCRを使ってスコアやレート、ベスト枠などを集計し、音ゲーのいろいろなことを便利にしてくれるDiscord Botです。
